About Dimple

Dimple is a simple discord bot created for basic tasks discord users may like to see in a bot.A lot of bots exist that can perform Dimple's functions but I made Dimple to create something of my own in a platform like discord which I use on a day to day basis. Dimple is based on the spirit from Mob Psycho that accompanied the protoganist Mob.

Dimple is able to:

  • Express how it feels through emojis
  • Give server info
  • Answer yes/no questions
  • Show roles available in a server
  • Can add suggestions to a seperate csv file
  • Can set a reminder in 24 hour time and within the day

Dimple's Origins

Dimple was made using discord.py, a python library for making discord applications. Dimple was created in the discord application itself. The bot token was taken and used for running the client. Many of the functions, that Dimple can perform, were made with the aid of the API's documentation. Dimple was made to enhance my understand of working with APIs and also to work on a fun project.
